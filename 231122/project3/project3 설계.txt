* 프로그램기능 

1. 운동종목 검색
-> 지난 5회차까지 했던 세트, 횟수, 무게 data 제공
// 개별운동종목 text에서 list의 length-4 ~ length의 data를 제공




2. 기록 입력
-> 운동종목, 기록(0set: 0kg / 0reps)
-> 오늘 운동 list에 추가됨
-> 개별 운동종목 list에 추가됨

3. 오늘
-> 오늘 운동 list 출력
(오늘 한 운동의 종목, 기록 값)

4. 저장
-> 오늘 운동 list를 text파일로 저장
    개별 운동종목 list의 text파일을 수정




* 구현기능
1. list class
- 오늘운동 list (오늘 한 운동종목들의 오늘 기록만 저장)
- 개별운동 list (특정 종목의 운동의 과거기록 저장)

2. GUI class
- 검색, 입력, 오늘, 저장의 button 만들기
- 검색한 운동종목의 기록data 보여주는 textfield
- 기록을 입력할 수 있는 textfield
- 오늘을 누르면 오늘 한 운동종목들, 기록 data를 보여주는 textfield

3.Event class
- button을 눌렀을 때 각 기능이 구현되도록
검색: 검색한 운동종목의 기록data를 보여줌
입력: 입력 시 data값을 오늘운동 list, 개별운동종목 list에 저장
오늘: 오늘 한 운동종목들의 기록 data를 보여줌
저장: list들을 text파일로 저장

4. main class
